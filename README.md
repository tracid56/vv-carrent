
### Manually
- Download https://github.com/Mika-fivem/vv-carrent
- Put it in the `[esx]` directory

## Installation
### Instal the resource
- Import `esx_carrent.sql` to your database
- Add `start vv-carrent` in your `server.cfg`:
```
start vv-carrent
```
